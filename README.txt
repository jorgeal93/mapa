# CampoGeo V6.8 — Carregamento corrigido

Correção principal:
- O carregamento/splash não fica mais infinito.
- Se alguma etapa demorar, o app abre o menu inicial mesmo assim.
- PDF.js carrega em segundo plano e não trava a abertura.
- Cache offline não prende o app.
- Avisos continuam aparecendo e sumindo sozinhos.
- Mantidos os ícones melhorados, GPS, alfinetes, cores, KMZ, limpar e offline.

Depois de subir:
1. Aperte Ctrl + F5 no PC para recarregar forte.
2. No celular, feche o app e abra de novo.
3. Se estiver no GitHub Pages, aperte Cache uma vez depois que abrir.
